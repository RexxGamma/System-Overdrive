# SYSTEM OVERDRIVE

___________________________________________________________________________________________________________________
# Description

System Overdrive is an top-down 2D shooter inspired by pew pew live and set in space whih has 8 diferent types of enemies and a maximum of 32 waves.

___________________________________________________________________________________________________________________
# How to Play?

After you click the "Play" button in the main menu,you will appear in the space,eventually the enemies of the wave 1 will appear near you.You should kill them all to win the wave,how many waves you can win?

# Controls

To move you can use the wasd keys or the arrows.W to move up,S to move down,A to move left and D to move right.

To attack you can use the left click or the Z key,the bullet will go to the mouse position

To dash you can use the left-shift key or the right click,the player will dash in the direction of the mouse too
